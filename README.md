## Como usar

Edite os arquivos Markdown e mantenha o historico pelo Git.

```bash
git add .
git commit -m "docs(curriculo): atualizar perfil profissional"
```

Quando o repositorio for publicado no GitHub, o GitHub Actions podera gerar artefatos em PDF/HTML a partir dos arquivos Markdown.
